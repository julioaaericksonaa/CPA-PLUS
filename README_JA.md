# CPA-PLUS

このリポジトリは、CLIProxyAPI と CPA-Manager-Plus を統合したプライベート運用向けビルドです。

最新の利用方法、保守方法、更新手順は以下を参照してください。

- `README.md`
- `README_CN.md`
- `docs/cpa-plus-work-log-and-maintenance.md`
- `docs/local-upstream-sync.md`

主要な入口：

```text
http://host:8317/management.html
```

注意：実運用の `config.yaml`、`.env`、`auths/`、`data/`、`logs/`、Token、Cookie、ローカル認証情報を GitHub にコミットしないでください。
